# Minna Journey — Master Quote Card Set T1–T18 v1.0
## Locked working set for track-level share cards

**Document status:** Locked  
**Owner:** Clinical / Product Director  
**Purpose:** ใช้เป็นชุดอ้างอิงหลักของ quote card แบบ track-level สำหรับ T1–T18 โดยยึดหลักว่า share card = 1 ใบ / 1 track completion, ใช้ B4 เป็น determinant, และ public-facing wording ต้องเป็น distilled resource ที่เสริม self-worth โดยไม่วินิจฉัยผู้เล่น

---

## Core rules

- Share card = 1 ใบ / 1 track completion
- Determinant = B4 choice
- good / else / stuck เป็น logic ภายในเท่านั้น ผู้เล่นไม่เห็น label นี้
- share-safe quote ต้องนุ่มกว่าข้อความในเกม
- ทุก state share ได้อย่างมีศักดิ์ศรีเท่ากัน
- quote card ต้องสะท้อนสิ่งที่ protagonist เห็นได้ / ทำได้ / ถือกลับไปได้
- quote card ต้องไม่ฟังเหมือนผลประเมินผู้เล่น

## Implementation note for T1/T2

T1 และ T2 เป็น old-schema tracks ที่ B4 ไม่มี stuck mapping ตรงแบบ track รุ่นใหม่  
ดังนั้น stuck share card line ของ 2 track นี้ใช้การสังเคราะห์จาก protagonist arc + badge stuck reference  
จนกว่าจะมี schema migration ในภายหลัง
