# Minna Journey — Pilot Quote Cards v1.0
## Gold Standard Reference Set for Share Card Language

Document status: Locked
Owner: Clinical / Product Director
Purpose: ใช้เป็น benchmark language set สำหรับ quote card ทุก track
ทุก quote ที่เพิ่มใหม่ต้องผ่านกรองด้วย tone และ structure ของชุดนี้

---

## หลักการ 3 ชั้นภาษา

ชั้นที่ 1 — choice / badge popup: ใช้ระหว่างเล่น ใกล้ความรู้สึกจริง ยังแชร์ไม่ได้
ชั้นที่ 2 — summary: สรุป arc ของ protagonist
ชั้นที่ 3 — quote card: public-facing, คม สั้น เท่ ปลอดภัย เสริม self-worth

badge text = immediate reflection
quote card = distilled resource

---

## State Design Intent

good = resource เชิง agency / clarity / grounded choice
else = resource เชิง permission / emerging awareness / kindness toward uncertainty
stuck = resource เชิง containment / self-compassion / permission to not be ready / dignity in unfinishedness

stuck ต้องไม่ฟังเหมือนรายงาน state ของเด็ก
ต้องฟังเหมือน resource ที่ประคองเด็กในจุดที่ยังไม่พร้อม

---

## Satir Belief Filter

ทุก quote card ต้องผ่าน 5 ข้อ:
1. คนมีพื้นฐานที่ดี
2. coping ไม่ใช่ความผิด แต่เป็นการเอาตัวรอด
3. ทุกคนมี resource ภายใน
4. การเปลี่ยนแปลงเป็นไปได้เสมอ
5. เราไม่จำเป็นต้องกดดันตัวเองให้หายทันที

---

## Gold Standard Set

### T10 — ห้องของมินนา 🖼️
Theme: Reflection & Self-awareness

Good: เห็นการเปลี่ยนแปลงรอบตัว — แล้วเริ่มเชื่อว่าเราเองก็เติบโตในจังหวะที่พอดีได้เหมือนกัน
Else: เห็นคนอื่นชัดขึ้น — และค่อยๆ หันกลับมามองความสวยงามในแบบของตัวเอง
Stuck: ยังไม่ต้องรู้ทุกอย่างในตอนนี้ — แค่เริ่มหันกลับมามองตัวเองก็นับเป็นจุดเริ่มต้นที่ดีมากแล้ว

### T15 — คืนที่บ้านมินนา 🌸
Theme: Gratitude & Belongingness

Good: กล้าที่จะรับความปรารถนาดี — และอนุญาตให้หัวใจได้สัมผัสถึงความอบอุ่นที่ได้รับ
Else: คืนนี้อาจยังไม่ครบทุกคำ — แต่ความอบอุ่นบางอย่างก็เกิดขึ้นแล้ว
Stuck: บ้านไม่จำเป็นต้องสมบูรณ์แบบเสมอไป — แค่เรายังพยายามหันหน้าเข้าหากันก็นับว่าเพียงพอ

### T17 — วันที่โมเน่รู้แล้ว 👟
Theme: Academic pressure / Self-direction

Good: ฟังเสียงคนอื่นได้ — แต่สุดท้ายเรามีสิทธิ์ที่จะเลือกเส้นทางที่เป็นของตัวเองจริงๆ
Else: ข้างในเริ่มชัดขึ้นแล้ว — แค่รอวันที่หัวใจพร้อมพูดออกมา
Stuck: บางครั้งเราต้องใช้เวลา — เพื่อแยกเสียงความคาดหวังของคนอื่นออกจากเสียงของตัวเราเอง

### T18 — เช้าที่ใบบัวเห็นยาย 🍵
Theme: Ambiguous loss / Acceptance

Good: นี่ไม่ใช่การยอมแพ้ — แต่คือการเลือกหยุดต่อสู้กับสิ่งที่เปลี่ยนไม่ได้ เพื่อกลับมาดูแลใจตัวเอง
Else: บางความจริงอาจยังยอมรับยาก — แต่การเริ่มมองเห็นโดยไม่หลบตา ก็นับเป็นจุดเริ่มต้นที่มีความหมายแล้ว
Stuck: ไม่ต้องรีบเข้าใจทุกอย่างในวันนี้ — การอนุญาตให้ตัวเองยังไม่พร้อม คือความใจดีอย่างหนึ่ง
